# LGTM# raycast-lgtm

Raycast用のextensionです。package managerはniを使ってます。

- Raycast： https://www.raycast.com/
- ni: https://github.com/antfu/ni

## 使用方法

クローンしてビルド
```
git clone git@github.com:mr04vv/raycast-lgtm.git
cd raycast-lgtm
ni  # yarn installとかでもいける？
nr build # yarn build?
```
